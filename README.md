# OLYMPIA GYM - Final Project PPW 🏋️‍♂️

Proyek ini adalah tugas Final Project untuk mata kuliah **Pengenalan Perancangan Web (SI05)** di Program Studi Sistem Informasi, Universitas Amikom Yogyakarta. 

Website ini merupakan prototipe dari *company profile* dan layanan untuk sebuah pusat kebugaran (Gym & Fitness Center) yang dibangun secara statis menggunakan *Client-Side Scripting*.

## 🛠️ Teknologi yang Digunakan
Sesuai dengan ketentuan Final Project, website ini dikembangkan tanpa *template* instan dan menggunakan:
* **HTML5:** Sebagai kerangka utama (minimal 5 halaman).
* **CSS3 & Bootstrap 5:** Untuk desain antarmuka yang modern dan responsif.
* **JavaScript (Vanilla JS):** Untuk memberikan interaktivitas pada elemen web (minimal 3 blok kode).

## 📂 Struktur Halaman
Website ini terdiri dari 5 halaman utama yang saling terhubung:
1. **`index.html` (Home):** Halaman pendaratan utama yang memuat gambar *hero* dan fitur interaktif Kalkulator BMI.
2. **`about.html` (About):** Halaman profil Gym yang memuat video *tour* fasilitas (menggunakan *iframe* YouTube).
3. **`classes.html` (Classes):** Halaman berisi tabel jadwal kelas mingguan secara terstruktur.
4. **`membership.html` (Membership):** Halaman daftar paket harga yang memuat *hyperlink* pendaftaran.
5. **`contact.html` (Contact):** Halaman layanan pelanggan yang memuat elemen *Form* untuk mengirim pesan.

## ✨ Fitur JavaScript
Terdapat 3 implementasi JavaScript utama pada proyek ini:
1. **Toggle Dark/Light Mode:** Fitur untuk mengubah tema warna website (berlaku global).
2. **Kalkulator BMI:** Fitur penghitung Indeks Massa Tubuh di halaman utama lengkap dengan validasi status (Kurus, Normal, Obesitas).
3. **Validasi Form Alert:** Fitur pada halaman *Contact* yang mencegah *reload* halaman dan memunculkan *alert custom* yang menyapa nama *user*.
